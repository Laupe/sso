# setOnLoginRequest( )

Listens for $_GET parameters and performs appropriate commands.

Listens for $_GET parameters and performs appropriate commands.

If email and password are set in $_GET, method creates SSO cookie and redirects user with generated token. If continue key is set in $_GET, method updates SSO cookie and redirects user with generated token. If relog key is set in $_GET, method shows login form. If none of conditions mentioned above is met, method checks if SSO cookie is set and tries to obtain user, otherwise method shows login form.
